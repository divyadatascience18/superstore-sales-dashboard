# Superstore Sales Dashboard (Excel)

## Overview
This project is an interactive Excel dashboard built using Superstore sales data.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPIs

## Key Insights
- Total Sales KPI
- Average Monthly Sales (dynamic with slicers)
- Best Performing Region
- Sales trend by month and year
- Top 10 products by sales
- Category-wise sales distribution

## Features
- Year, Region, Segment, and Category slicers
- Fully dynamic KPIs
- Professional dashboard layout

## Dataset
Superstore Sales Dataset (commonly used for analytics practice)
